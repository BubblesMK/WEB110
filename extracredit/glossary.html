HTML and CSS Glossary

Chapter 1 - Internet and Web Basics

Hyperext Markup Language (HTML)
Is the set of markup symbols or codes placed in a file that is intended for display on a web page.
Document Type Definition (DTD)
Identifies the verision of HTML being used in the web page. The DTD for HTML5 is <!DOCTYPE html>.
File Transfer Protocol (FTP)
A set of rules that allows files to be exchanged between computers on the Internet.
Protocols
Rules that describe how clients and servers communicate with each other over a network.
Section 508 of the Federal Rehablitation Act
This law requires developers creating information technology including web pages for use by the federal government to provide for accessibility.

Chapter 2 - HTML Basics

Heading Element
Organized into six levels: h1 through h6. H1 is the largest text and H6 is the smallest text.
Paragraph Element
Groups sentences and sections of text together.
Line Break Element
Causes the browser to advance to the next line before displaying the next elemenet and is coded <br>.
Phrase Element
Indicates the context and meaning of the text between the container tags.
HTML Validation
Provides you with quick self-assessment which allows you to prove that your syntax is correct.

Chapter 3 - Web Design Basics

Target Audience
The people who will use your site. For example, children, young and older adults, college students, or young couples.
Site Map
A diagram of the organization of a website. There are three common types. Hierarchical, Linear and Random.
Hierarchical Organization
Characterized by a clearly defined home page with links to major site sections.
Image Optimization
Process of ceating an image with the lowest file size that still has a good image quality.
Wireframe
A sketch or diagram of a web page that shows the structure of basic page elements such as the header, navigation, content area, and footer.

Chapter 4 - Cascading Style Sheets Basics

Background-Color
Property to configure the background color of an element.
Class Selector
Applies a CSS declaration to one or more areas on a web page.
ID Selector
Identifies and applies a CSS declaration to a single area on a web page.
Span Element
Defines a section on a web page that is displayed inline without empty space above and below.
CSS Validation
Serves as a tool to check the syntax of your CSS code and ensure it adheres to web standards.

Chapter 5 - Graphics & Text Styling Basics

Image Optimization
The process of creating an image with the lowest file size that still renders good quality image.
Lossless compression
Means nothing in the original image is lost and the compressed image will contain the same amount of pixels as the original image.
Lossy compression
Some pixels in the original image are lost or removed from the compressed file. When displayed in the browser the image will be similar to the original image but not exactly the same.
Background Property
Configures multiple background images.

Chapter 6 - More CSS Basics

Width Property
Configures the width of an element's content in the browser viewport with either a numeric value unit such as px or em or percentage.
Box Model
A box that consist of a content area surrounded by padding, a border, and margins.
Border property
Configures the border, or boundary, around an element.
Opacity Property
Configures the transperancy of an element.
Background-size Property
Used to resize or scale the background image.

Chapter 7 - Page Layouts Basics

Normal Flow
Displays the elements on the page in the order they appear in the web page.
Overflow Property
Used to clear a float.
Box-Sizing Property
The browser calculation of the width or height to include the content's actual width or height in addition to the width or height of any padding and border that may exist.
Pseudo-class
Applies a special effect to a selector. There are 5 classes that can be applied to the anchor element. They are as followed:
  • :link
  • :visited
  • :focus
  • :hoover
  • :active
Position Property
Configures the type of position used when the browser renders an element. Examples of position properties are static, fixed, relative, sticky, and absolute.

Chapter 8 - Responsive Layout Basics

Flexible Box Layout (flexbox)
Provides a flexible layout. Elements contained within the container can be configured in one dimension horizontally or vertically.
Flex-Wrap Property
Configures whether flex items are displayed on multiple lines.
Flex-Direction Property
Configure the flow direction of the flex item either in a row (horizontally) or in a column (vertically).
CSS Grid Layout
Configures a two-dimensional grid-based layout.
Grid-Area Property
Associates a grid item with a named area of the grid such as header, main, nav and footer.
Feature Query
A conditional that can be used to test for support of a CSS property and if support is found, apply the specified style rules. The feature query is coded @support() rule.

Chapter 10 - Form Basics

Common Gateway Interface (CGI)
A protocol, or standard method, for a web server to pass a web page user's request which is initiated through the use of a form.
Server-Side Scripting
A technology in which a Server-Side script is run on a web server to dynamically generate web pages.
Label Element
A container tag that associates a text description with a form control.
Form Element
Contains a form on a web page.
Input Element
Configures a wide variety of different form controls, including text boxes, submit buttons, check boxes, radion buttons, date and time pickers, and sliders.
Legend Elements
Provides a description, which can include heading tags, for fiedset grouping.
